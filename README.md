# MingxuanGame 黑历史合集

本仓库收集了我早期（即 2022 年以前）编写的代码。

## 警告

本仓库的代码包含了很多“血压升高”的行为，包括但不限于：

* 拼音变量名
* 不遵守 PEP8 代码规范
* 使用 CQ 码
* 异步里使用同步代码
* 不完整的 Type Hints
* 塑料英文与机翻
* ...

## 指南

文件夹内一般都具有 `README.md`，如果文件夹曾是一个 Git 仓库，那么文件夹里会包含 `COMMITS.md` 和 `.git.zip`。

`COMMITS.md` 存储了该仓库的提交摘要（使用 `git log` 查看的）

`.git.zip` 即 `.git` 文件夹，可解压。
